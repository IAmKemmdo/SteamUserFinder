## Opis projektu
Aplikacja umożliwia wyszukiwanie profili użytkowników Steam na podstawie ich SteamID64 lub customowym url/id.  
Wyświetla podstawowe informacje o użytkowniku i możliwość zobaczenia jego zdjęcia profilowego w pełnej rozdzielczości

---

# Funkcjonalności

 ###  **-Wyszukiwanie użytkownika po SteamID64 lub customowym id, co pozwoli zobaczyć jego:**
- **Nick**  
- **Prawdziwe imię użytkownika**  
- **Kraj pochodzenia użytkownika**  
- **Data założenia konta**  
- **Aktualny status**  
- **Aktualnie grana gra** (lub *Nic*, jeśli użytkownik w nic nie gra)  
- **Podgląd pełnego avatara w osobnej podstronie**  
---
## Instalacja i uruchomienie projektu (npm):

1.Sklonuj repozytorium
```
git clone https://github.com/IAmKemmdo/SteamUserFinder.git
```
2.Zainstaluj wymagane moduły
```
npm install
```
3.Uruchom aplikację
```
npm run dev
```
## Instalacja i uruchomienie projektu (bun):

1.Sklonuj repozytorium
```
git clone https://github.com/IAmKemmdo/SteamUserFinder.git
```
2.Zainstaluj wymagane moduły
```
bun install
```
3.Uruchom aplikację
```
bun run dev
```
